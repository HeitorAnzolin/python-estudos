Programa simples de terminal feito em Python que permite adicionar, visuzalizar e remover tarefas de uma lista.

Este projeto foi criado como exercício de apredizado em Python.

Funcionalidades: 
    - Adicionar tarefas
    - Visualizar tarefas
    - Remover tarefas
    - Menu interativo no terminal
    - Limpeza da tela entre operações

Tecnologia utilizadas:
    - Python 3

Como executar:
    1. Clone o repositório: git clone https://github.com/seu-usuario/nome-do-repositorio.git

    2. Entre na pasta do projeto: cd nome-do-repositorio

    3. Execute o programa: python nome_do_arquivo.py
